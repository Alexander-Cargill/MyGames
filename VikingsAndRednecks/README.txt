
While in the directory with the VikingsAndRednecks.py file, do the
following steps:

1. Ensure you have python installed on your machine:
	python --version

2. Make sure you have Pygame installed
	pip install pygame

3. Run the game
	python VikingsAndRednecks.py



Game Instructions: 

To play as Ragnar, the viking, you need to use the WASD keys and to play as Lennie, the Redneck, you need to use the arrow keys.
To  shoot the gun as Ragnar, press e. To shoot the gun as Lennie, press l.
To win use the powerups to defeat the other player.
