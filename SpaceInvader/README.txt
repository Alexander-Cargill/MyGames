
While in the directory with the VikingsAndRednecks.py file, do the
following steps:

1. Ensure you have python installed on your machine:
	python --version

2. Make sure you have Pygame installed
	pip install pygame

3. Run the game
	python SpaceInvader.py



Game Instructions: 

Press space to shoot. Use the A and D keys to shoot the aliens.
The levels get progressively harder!
